Car Make Explorer

This page uses a public car API from NHTSA to load a list of vehicle manufacturers. Pick a make on the right and then click View Models to see all models for that brand.

  • Uses a REST API with JSON data.
  • HTML, Bootstrap, and vanilla JavaScript.
  • Responsive layout for phones, tablets, and desktops.

Tip: If the list does not load, check your internet connection and refresh the page.
